Why Enums?

Frank G.

In what situation whould I use Enums over normal variables like ints or strings?

Dominic C.

Enums have the following advantages:

  • Compared to using standalone int variables, enums group all names under a single namespace
  • Unlike strings, enums can be used in switch statements
  • Enums are good for saving space, as there is no need to make multiple variables (which can be a problem when using strings)
